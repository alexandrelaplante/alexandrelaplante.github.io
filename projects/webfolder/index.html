# webfolder.js ## Description webfolder.js is an easy way to let you host a folder as a website using client side templating. This site, alexandre.ml, uses it. It allows you to define a template (named base.html) that all pages will use, and lets you write your individual pages in markdown. Any changes to base.html are reflected in every page of the site. webfolder.js simply gets base.html through ajax and styles the page dynamically on the client side. If you're making any serious website, use a real server side templating engine. ## Usage Include webfolder.js and a file named 'base.html' at the top level of your site. Use the `<html>` technique again for this.
